## Datasets
### Image Datasets
-[DigitFive](https://ai.bu.edu/M3SDA) composed by MNIST, MNISTM, SVHN, USPS, and SythDigits 
-[Amazon Review](https://nijianmo.github.io/amazon/index.html) composed by Books, DVDs, Electronics, and Kitchen housewares
